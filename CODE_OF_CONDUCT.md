# Code of Conduct

## Our Pledge

We are committed to making participation in this project respectful, practical, and focused on improving the skill for coding workflows.

All contributors and maintainers are expected to:

- Use welcoming and inclusive language.
- Respect different experience levels and project constraints.
- Give feedback that is specific, actionable, and tied to the work.
- Keep disagreements technical and evidence-based.
- Avoid harassment, personal attacks, private-information disclosure, and other conduct that makes participation unsafe.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, reviews, and any community channel used for this repository.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code of conduct.

If you need to report unacceptable behavior, contact the maintainers through the repository owner or the private security/reporting channel listed in `SECURITY.md` when appropriate. Reports should include links, screenshots, dates, and any relevant context. Maintainers should handle reports with care and limit disclosure to people who need the information to respond.
