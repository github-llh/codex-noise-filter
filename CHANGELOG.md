# Changelog

All notable changes to `codex-noise-filter` are recorded here.

## 0.4.0 - 2026-06-15

- Expanded `SKILL.md` trigger description with higher-signal coding keywords while keeping the scope limited to programming tasks.
- Added README sections for Why Use It, Before/After positioning, and copyable activation prompts in Chinese and English.
- Added five scenario examples under `examples/` for Java backend, Maven multi-module, Vue/React, Python, and Mini Program work.
- Added team rollout templates under `templates/` for lightweight global AGENTS guidance, repository installation, and activation checks.

## 0.3.0 - 2026-06-14

- Added indexed rules for Python, Vue/React, Mini Program, environment discovery, and context management.
- Kept personal machine paths out of reusable documentation and routed environment discovery through `.codex/local-environment.json`.
- Added Apache-2.0 licensing and bilingual README coverage.

## 0.2.0 - 2026-06-14

- Strengthened Java backend layering rules for Controller, Service interface, Service implementation, DTO/VO/entity comments, Lombok, validation, transactions, concurrency, and abstraction.
- Clarified that rules apply to both new code and existing touched code.

## 0.1.0 - 2026-06-14

- Created the indexed `SKILL.md` + `references/00-index.md` + topic references structure.
- Added hard gates for context narrowing, call-chain confirmation, minimal edits, validation, and Context Capsule delivery.
